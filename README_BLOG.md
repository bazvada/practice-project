## Приложение blog

Приложение блога для проекта `practice-project`.
Это приложение было добавлено в проект, созданный на основе [`wemake-django-template`](https://github.com/wemake-services/wemake-django-template).  
Проект запускается локально с помощью Docker Compose и использует PostgreSQL в отдельном контейнере.

## Описание

В приложении `blog` реализована простая работа с записями блога через веб-интерфейс.

Модель `Post` содержит поля:

- `title` — заголовок записи
- `text` — текст записи
- `image` — изображение записи
- `created_at` — дата создания записи

Для работы с записями блога реализованы основные CRUD-операции: 
- просмотр списка записей, 
- просмотр отдельной записи, 
- создание новой записи,
- редактирование записи,
- удаление записи.

## Требования для запуска

Для запуска проекта потребуется:
- установленный и запущенный Docker Desktop
- Docker Compose
- локальная копия проекта `practice-project`

Версии Python и PostgreSQL задаются настройками проекта и Docker Compose:

- `python3.13`
- `postgresql`
- актуальная версия `docker`

## Разработка и запуск

Для локального запуска проекта используется Docker Compose.

Запуск контейнеров: docker compose up -d
Остановка контейнеров: docker compose down
Просмотр логов Django-приложения: docker compose logs web
Проверка конфигурации Docker Compose: docker compose config
Выполнение Django-команд внутри контейнера web: docker compose exec web python manage.py ...

## База данных

Применение миграций: docker compose exec web python manage.py migrate
Создание суперпользователя: docker compose exec web python manage.py createsuperuser

## Страницы приложения

Приложение доступно по адресу: http://localhost:8000/blog/

Страницы приложения:
/blog/ - список записей
/blog/<id>/ - отдельная страница записи
/blog/create/ - создание записи
/blog/<id>/edit/ - редактирование записи
/blog/<id>/delete/ - удаление записи

Админ-панель Django: http://localhost:8000/admin/

## Реализованные представления

Для работы используются:

ListView — отображение списка записей
DetailView — отображение отдельной записи
CreateView — создание записи
UpdateView — редактирование записи
DeleteView — удаление записи

## Шаблоны

Для работы приложения были созданы следующие шаблоны:

post_list.html — список записей

post_form.html — форма создания и редактирования записи
