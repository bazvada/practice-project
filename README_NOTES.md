## Приложение notes

Приложение заметок для проекта `practice-project`.
Это приложение было добавлено в проект, созданный на основе [`wemake-django-template`](https://github.com/wemake-services/wemake-django-template).  
Проект запускается локально с помощью Docker Compose и использует PostgreSQL в отдельном контейнере.

## Описание

В приложении `notes` реализована простая работа с заметками через веб-интерфейс.

Модель `Note` содержит поля:

- `title` — заголовок заметки
- `text` — текст заметки
- `is_important` — отметка важности заметки

Для работы с заметками реализованы основные CRUD-операции: 
- просмотр списка заметок, 
- просмотр отдельной заметки, 
- создание новой заметки,
- редактирование заметки,
- удаление заметки.

## Требования для запуска

Для запуска проекта потребуется:
- установленный и запущенный Docker Desktop
- Docker Compose
- локальная копия проекта `practice-project`

Версии Python и PostgreSQL задаются настройками проекта и Docker Compose:

- `python3.13`
- `postgresql`
- актуальная версия `docker`

## Разработка и запуск

Для локального запуска проекта используется Docker Compose.

Запуск контейнеров: docker compose up -d
Остановка контейнеров: docker compose down
Просмотр логов Django-приложения: docker compose logs web
Проверка конфигурации Docker Compose: docker compose config
Выполнение Django-команд внутри контейнера web: docker compose exec web python manage.py ...

## База данных

Применение миграций: docker compose exec web python manage.py migrate
Создание суперпользователя: docker compose exec web python manage.py createsuperuser

## Страницы приложения

Приложение доступно по адресу: http://localhost:8000/notes/

Страницы приложения:
/notes/ - список заметок
/notes/<id>/ - отдельная страница заметки
/notes/create/ - создание заметки
/notes/<id>/edit/ - редактирование заметки
/notes/<id>/delete/ - удаление заметки

Админ-панель Django: http://localhost:8000/admin/

## Реализованные представления

Для работы используются:

ListView — отображение списка заметок
DetailView — отображение отдельной заметки
CreateView — создание заметки
UpdateView — редактирование заметки
DeleteView — удаление заметки
